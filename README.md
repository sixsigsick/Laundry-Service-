# 🧺 Laundry Service System

## 📌 Project Overview
ระบบจัดการร้านซักรีด (Laundry Service System) เป็นเว็บแอปพลิเคชันที่ช่วยให้ลูกค้าและผู้ดูแลระบบสามารถจัดการบริการซักผ้าได้อย่างสะดวก  
รองรับทั้งการใช้งานของผู้ใช้ทั่วไป (User) และผู้ดูแลระบบ (Admin)

---

## 🎯 Objectives
- เพื่อพัฒนาระบบจัดการร้านซักรีดแบบออนไลน์
- ลดความผิดพลาดในการจัดการออเดอร์
- เพิ่มความสะดวกในการติดตามสถานะงาน
- ช่วยให้ผู้ดูแลสามารถบริหารรายได้และลูกค้าได้ง่ายขึ้น

---

## 🚀 Features

### 👤 User (ลูกค้า)
- สมัครสมาชิก / Login
- สร้างรายการซักผ้า
- ดูราคารวมของออเดอร์
- ดูสถานะออเดอร์
- ดูประวัติการใช้บริการ
- ยกเลิกออเดอร์

---

### 🛠 Admin (ผู้ดูแลระบบ)
- ดูรายการออเดอร์ทั้งหมด
- อัปเดตสถานะออเดอร์ (เช่น กำลังซัก / เสร็จแล้ว)
- เพิ่ม / แก้ไข / ลบ ประเภทบริการ + ราคา
- ดูรายชื่อลูกค้า
- ดูรายงานรายได้
